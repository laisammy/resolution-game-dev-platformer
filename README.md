# Platformer Game
# Made for Hack Club's Resolution Game Dev Week 5
This is a simple platformer game made with Godot 4.6.1. Players can type words (using random word API) with their keyboard in order to give their character a limited time jump boost.
A simple UI is used to track the time remaining for the current jump boost and the amount of coins collected.

# Controls
Arrow keys to move
Space to jump
